# OLA Data Analysis Dashboard

This project is an interactive OLA Data Analysis Dashboard developed in Power BI to analyze ride booking performance and generate meaningful business insights from transportation data. The dashboard focuses on key operational metrics such as total bookings, successful rides, cancellations, revenue, customer trends, and driver performance, helping to understand the overall efficiency of ride operations.

The main objective of this project is to transform raw ride-booking data into a clear and interactive dashboard that supports data-driven decision-making. Using Power BI, the data was cleaned, modeled, and visualized through KPIs, charts, filters, and reports to identify important patterns such as booking trends, cancellation reasons, revenue distribution, customer demand, and ride performance.

This project demonstrates my ability to work on real-world business datasets, build professional dashboards, and present insights in a visually appealing and easy-to-understand format. It also highlights my skills in data cleaning, data modeling, DAX, dashboard design, business analysis, and data visualization.
